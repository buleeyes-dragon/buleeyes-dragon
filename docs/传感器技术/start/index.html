绪论​

1.1 传感器的定义​

定义
  • 传感器：[非电量->电量]---将各种非电量按一定规律转换成便于处理和传输的电量（如电阻，电压，电流）的装置。
  • Sensor 与Actuator：执行器，电量转化为非电量

传感器技术的概念​

1.3.1 被测量的分类​

分类
  • 1.电量： 电压、电流、功率、电阻、电容、电感
  • 2.非电量：
    • (1)机械量: 位移、振动、速度、加速度、力、力矩
    • (2)热工量: 温度、压力、流量
    • (3)化工量: 浓度、成分、湿度等
